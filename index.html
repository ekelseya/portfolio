Some of my favorite projects

The Cage Rage Rater

A full-stack web application to rate Nicolas Cage's direct-to-video movies of the last decade built with Django Rest Framework and React.js. I focused on increasing my understanding of functional components and React hooks. I'm also having fun working through each movie! The DRF API is deployed on Heroku and the web app is deployed on Google Firebase. Due to the nature of free-tier hosting plans, please give the servers a moment to wake up.

  • React
  • Django Rest Framework

RESTful Blog

A blog application built with Node, Express, and MongoDB. It uses passport for simple authentication so users can log-in and leave comments. I used Fomantic UI (the community fork of Semantic UI) for styling. Only admin users can create new posts. HTML mark up is allowed in new posts, but it is sanitized before being saved to the MongoDB database. Anyone can create a user account and comment on posts. Users can edit and delete their own comments, but only admin users can delete or edit any comment.

  • Node
  • Espress
  • MongoDB
